# <MasterChip value={"GET"} /> grantRefreshToken

## API Details

- **Description** If refresh token sent transparently via Http cookie is valid, renew a refresh token
- **Version** `v1`
- **Allowed Connection Types** `rpc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** refresh
- **Flags** Log message
- **Scope** global


## Request

### Signal format

```json
{
  "refreshToken": "string",
  "callerId": "UserId",
  "callerDeviceId": "DeviceId",
  "clusterToken": "string",
  "rememberMe": "boolean"
}
```
