# entSpreadsheetRowsGet

## API Details

- **Description** Get spreadsheet rows for editing
- **Version** `v1`
- **Method** `post`
- **Allowed Connection Types** `wsoc` `rpc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Scope** globalOrEnt


## Request

### Message format

```json
{
  "spreadsheetId": "MetaIdSpreadsheet",
  "actionId": "MetaIdAction",
  "searchText?": "string",
  "orderByFieldId?": "MetaIdField",
  "groupByFieldId?": "MetaIdField",
  "ascOrder?": "boolean",
  "categoryFieldIds?": [
    "MetaIdField"
  ],
  "advancedFieldIds?": [
    "MetaIdField"
  ]
}
```

### Signal format

```json
{
  "rowIdSet?": [
    "RowId"
  ],
  "groupByRowIdSetMap?": {
    "string": [
      "RowId"
    ]
  },
  "outputFormId": "MetaIdForm"
}
```
