# spreadsheetRowSend

## API Details

- **Description** Sends a form message
- **Version** `v1`
- **Method** `post`
- **Allowed Connection Types** `rpc` `wsoc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Publishes** `newMessage` `recent` `lastMessage` `badge` `mediaList` `callerBadgeMap` `callerRecentList` `formResult` `spreadsheetRowExpiry` 
- **Scope** ent
