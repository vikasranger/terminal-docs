# entSpreadsheetRowShare

## API Details

- **Description** Generate Spreadsheet row Url to Share
- **Version** `v1`
- **Method** `get`
- **Allowed Connection Types** `wsoc` `rpc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Scope** ent
