# spreadsheetRowGet

## API Details

- **Description** Gets a form result
- **Version** `v1`
- **Method** `get`
- **Allowed Connection Types** `rpc` `wsoc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Scope** ent


## Request

### Message format

```json
{
  "rowId": "RowId",
  "spreadsheetId": "MetaIdSpreadsheet",
  "version?": "string"
}
```
