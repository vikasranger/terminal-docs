# spreadsheetRowAutomationStateGet

- **Description** Gets row's running automations list
- **Version** `v1`
- **Method** `get`
- **Allowed Connection Types** `rpc` `wsoc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Scope** ent
