# entReportShare

## API Details

- **Description** Generate Report Url to Share
- **Version** `v1`
- **Method** `put`
- **Allowed Connection Types** `wsoc` `rpc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Scope** globalOrEnt
