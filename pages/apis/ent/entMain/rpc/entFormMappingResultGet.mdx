# entFormMappingResultGet

## API Details

- **Description** Map one form value to another
- **Version** `v1`
- **Method** `post`
- **Allowed Connection Types** `wsoc` `rpc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Scope** ent
