# agentActiveStatusUpdate

## API Details

- **Description** Add and remove agent from active agent list
- **Version** `v1`
- **Method** `put`
- **Allowed Connection Types** `wsoc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Scope** global

### Message format

```json
{
  "entId": "EntId",
  "active": "boolean"
}
```
