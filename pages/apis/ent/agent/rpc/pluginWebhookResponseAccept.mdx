# pluginWebhookResponseAccept

## API Details

- **Description** Accept webhook response
- **Version** `v1`
- **Method** `post`
- **Allowed Connection Types** `rpc` `wsoc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Scope** global
