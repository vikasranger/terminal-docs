# studioSearch

## API Details

- **Description** Search all the artifacts and their users/admin
- **Version** `v1`
- **Method** `get`
- **Allowed Connection Types** `rpc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Scope** global


## Request

### Message format

```json
{
  "searchId": "string",
  "query": "string",
  "filterArtifactIdSet?": [
    "ArtifactId"
  ]
}
```

### Signal format

```json
{
  "searchResultMap": {
    "ArtifactId": {
      "searchResult?": {

            "string"
```
