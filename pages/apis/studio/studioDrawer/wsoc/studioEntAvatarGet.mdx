# studioEntAvatarGet

## API Details

- **Description** Get enterprise avatar for studio from draft or deployed copy
- **Version** `v1`
- **Method** `get`
- **Allowed Connection Types** `rpc` `wsoc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Scope** ent

### Message format

```json
{
  "version?": "string"
}
```
