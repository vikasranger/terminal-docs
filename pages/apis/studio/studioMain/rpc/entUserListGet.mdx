# entUserListGet

## API Details

- **Description** Get enterprise users for studio
- **Version** `v1`
- **Method** `get`
- **Allowed Connection Types** `rpc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Scope** ent
