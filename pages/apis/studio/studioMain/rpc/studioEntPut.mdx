# studioEntPut

## API Details

- **Description** Update existing enterprise draft
- **Version** `v1`
- **Method** `put`
- **Allowed Connection Types** `rpc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Publishes** `studioEnt` `studioEntAvatar` 
- **Scope** ent
