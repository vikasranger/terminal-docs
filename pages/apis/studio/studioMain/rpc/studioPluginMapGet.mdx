# studioPluginMapGet

## API Details

- **Description** Get caller's plugin map
- **Version** `v1`
- **Method** `get`
- **Allowed Connection Types** `rpc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Scope** global
