# pluginAdminCallerGet

## API Details

- **Description** Get caller admin details for plugin
- **Version** `v1`
- **Method** `get`
- **Allowed Connection Types** `rpc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Scope** plugin


## Request

### Signal format

```json
{
  "adminId": "AdminId",
  "nickName": "string",
  "doNotShowOptionSet?": [
    "admins"
  ],
  "doNotEditOptionSet?": [
    "admins"
  ],
  "hasLock?": "boolean"
}
```
