## API Details

- **Description** Get read and received receipts for given message
- **Version** `v1`
- **Allowed Connection Types** `rpc` `wsoc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Scope** globalOrEnt

### Message format

```json
{
  "offset": "number",
  "chatId": "ChatId"
}
```

### Signal format

```json
{
  "entUserReceiptMap": {
    "EntUserId": {
      "readOn?": "string",
      "deliveredOn?": "string"
    }
  }
}
```
