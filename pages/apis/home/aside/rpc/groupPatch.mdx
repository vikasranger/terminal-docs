# groupPatch

## API Details

- **Description** Updates group details like name, media avatar, configurations except membership
- **Version** `v1`
- **Method** `patch`
- **Allowed Connection Types** `rpc` `wsoc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Publishes** `groupMembership` `groupAvatar` 
- **Scope** global


## Request

### Message format

```json
{
  "name?": "string",
  "about?": "string",
  "mediaIdAvatar?": "MediaIdAvatar",
  "settings?": {
    "anyOneCanJoin": "boolean",
    "onlyAdminCanUpdateGroupInfo": "boolean",
    "onlyAdminCanSendMessages": "boolean"
  },
  "groupId": "GroupId",
  "patchPropNameSet": [
    "name | about | mediaIdAvatar | settings"
  ],
  "version?": "string"
}
```

### Signal format

```json
{}
```
