# messageListGet

## API Details

- **Description** Get messages for given chat. API supports pagination
- **Version** `v1`
- **Method** `get`
- **Allowed Connection Types** `rpc` `wsoc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Scope** globalOrEnt


## Request

### Message format

```json
{
  "chatId": "ChatId",
  "pageSize?": "number"
}
```
