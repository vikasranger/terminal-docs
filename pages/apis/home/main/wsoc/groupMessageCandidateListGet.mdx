# groupMessageCandidateListGet

## API Details

- **Description** Get group candidate list to send as group message
- **Version** `v1`
- **Method** `get`
- **Allowed Connection Types** `rpc` `wsoc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Scope** globalOrEnt
