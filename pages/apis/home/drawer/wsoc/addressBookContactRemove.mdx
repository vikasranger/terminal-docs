# addressBookContactRemove

## API Details

- **Description** Removes given contact contact from caller's address book by handle
- **Version** `v1`
- **Method** `delete`
- **Allowed Connection Types** `rpc` `wsoc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Publishes** `callerAddrBook` 
- **Scope** global


## Request

### Message format

```json
{
  "handle": "string"
}
```

### Signal format

```json
{}
```
