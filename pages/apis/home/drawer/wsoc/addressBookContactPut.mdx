# addressBookContactPut

## API Details

- **Description** Add new contact in caller's address book
- **Version** `v1`
- **Method** `put`
- **Allowed Connection Types** `rpc` `wsoc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Publishes** `callerAddrBook` 
- **Scope** global


## Request

### Message format

```json
{
  "nickName": "string",
  "handle": "string"
}
```

### Signal format

```json
{}
```
