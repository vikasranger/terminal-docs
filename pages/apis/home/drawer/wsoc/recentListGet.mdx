# recentListGet

## API Details

- **Description** Get items list for displaying in recents
- **Version** `v1`
- **Method** `get`
- **Allowed Connection Types** `wsoc` `rpc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Scope** global


## Request

### Message format

```json
{
  "filterEntIdSet?": [
    "EntId"
  ],
  "version?": "string"
}
```

### Signal format

```json
{
  "version?": "string",
  "recentList": [
    {
      "entId": "EntId",
      "chatId": "ChatId",
      "isPinned?": "boolean"
    }
  ]
}
```
