# groupAvatarGet

## API Details

- **Description** Get group details including its icon
- **Version** `v1`
- **Method** `get`
- **Allowed Connection Types** `rpc` `wsoc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Scope** globalOrEnt


## Request

### Message format

```json
{
  "groupId": "GroupId",
  "version?": "string"
}
```

### Signal format

```json
{
  "groupId": "GroupId",
  "entId": "EntId",
  "name": "string",
  "label?": "string",
  "about?": "string",
  "mediaIdAvatar?": "MediaIdAvatar",
  "isAdmin?": "boolean",
  "isMember": "boolean",
  "version": "string"
}
```
