# callerProfilePatch

## API Details

- **Description** Updates caller profile information
- **Version** `v1`
- **Method** `patch`
- **Allowed Connection Types** `rpc` `wsoc` 
- **Allowed Deployment Kinds** `beta` `prod` `localHost` `test` 
- **Allowed Device Types** `test` `ios` `android` `web` `agent` 
- **SLA** 600
- **Token** bearer
- **Flags** Log message
- **Publishes** `userAvatar` `caller` 
- **Scope** global

  "about?": "string",
  "mediaIdAvatar?": "MediaIdAvatar",
  "enterIsSendMobile?": "boolean",
