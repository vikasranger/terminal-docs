## ensure

This command accepts the following arguments:

**`doNotShareWithPublicLibrary`**
```js filename="type"
 null | true | false | yes | no
```
**`phrase`**
```js filename="type"
 null | @null | @empty | phrase
```
**`language`**
```js filename="type"
 null | @null | @empty | language
```
**`value`**
```js filename="type"
 null | @null | @empty | translationText
```
