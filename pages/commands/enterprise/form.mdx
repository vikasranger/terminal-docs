## update

This command accepts the following arguments:

**`name`**
```js filename="type"
 Symbol
```
**`label`**
```js filename="type"
 null | @null | @empty | label
```
**`description`**
```js filename="type"
 null | @null | @empty | paragraph
```
**`filterTags`**
```js filename="type"
 null | @null | @empty | text1, text2,...
```
**`chatLabelField`**
```js filename="type"
 null | Symbol
```
**`textPattern`**
```js filename="type"
 @null | @empty | @@text | text
```
**`chatPattern`**
```js filename="type"
 @null | @empty | @@paragraph | paragraph
```
**`tabVariant`**
```js filename="type"
 fullWidth | scrollable | standard
```
**`chatBubbleFields`**
```js filename="type"
 null | @empty | Field1, Field2,...
```
**`allowToPrintForm`**
```js filename="type"
 null | true | false | yes | no
```
