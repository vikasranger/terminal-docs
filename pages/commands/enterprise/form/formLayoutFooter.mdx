## update

This command accepts the following arguments:

**`showSeparator`**
```js filename="type"
 null | true | false | yes | no
```
**`backgroundColor`**
```js filename="type"
 @null | @empty | @@color | color
```
**`foregroundColor`**
```js filename="type"
 @null | @empty | @@color | color
```
**`line`**
```js filename="type"
 first | second | third
```
**`label`**
```js filename="type"
 first | second | third
```
**`textPattern`**
```js filename="type"
 @null | @empty | @@text | text
```
**`textFont`**
```js filename="type"
 @null | @empty | @@font | font
```
**`textOpacity`**
```js filename="type"
 @null | @empty | @@decimal | decimal
```
