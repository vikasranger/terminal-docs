## update

This command accepts the following arguments:

**`name`**
```js filename="type"
 null | Symbol
```
**`comp`**
```js filename="type"
 null | sectionField | grid/field
```
**`visibilityAction`**
```js filename="type"
 blink | click | disable | enable | hidden | highlight | invisible | shake | visible
```
