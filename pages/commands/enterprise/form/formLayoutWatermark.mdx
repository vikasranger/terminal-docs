## update

This command accepts the following arguments:

**`textPattern`**
```js filename="type"
 @null | @empty | @@text | text
```
**`textFont`**
```js filename="type"
 @null | @empty | @@font | font
```
**`textPosition`**
```js filename="type"
 @null | @empty | @@themePlacement | themePlacement
```
**`textOpacity`**
```js filename="type"
 @null | @empty | @@decimal | decimal
```
