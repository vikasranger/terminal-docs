## update

This command accepts the following arguments:

**`disabled`**
```js filename="type"
 null | true | false | yes | no
```
**`name`**
```js filename="type"
 Symbol
```
**`label`**
```js filename="type"
 null | @null | @empty | label
```
**`description`**
```js filename="type"
 null | @null | @empty | paragraph
```
**`filterTags`**
```js filename="type"
 null | @null | @empty | text1, text2,...
```
**`propertyEditorLabel`**
```js filename="type"
 @null | @empty | @@text | text
```
**`direction`**
```js filename="type"
 horizontal | vertical
```
**`sectionVariant`**
```js filename="type"
 form | propertyEditor
```
