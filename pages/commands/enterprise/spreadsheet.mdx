## update

This command accepts the following arguments:

**`name`**
```js filename="type"
 Symbol
```
**`label`**
```js filename="type"
 null | @null | @empty | label
```
**`description`**
```js filename="type"
 null | @null | @empty | paragraph
```
**`filterTags`**
```js filename="type"
 null | @null | @empty | text1, text2,...
```
**`searchableFields`**
```js filename="type"
 null | @empty | sectionField, grid/field...
```
**`queryableFields`**
```js filename="type"
 null | @empty | sectionField, grid/field...
```
