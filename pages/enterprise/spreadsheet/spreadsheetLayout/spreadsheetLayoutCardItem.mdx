## update

This command accepts the following arguments:

**`line`**
```js filename="type"
 fifth | first | fourth | second | third
```
**`imageCorner`**
```js filename="type"
 @null | @empty | @@imageCorner | imageCorner
```
**`images`**
```js filename="type"
 @null | @empty | null | GridFieldOrVar, Section/FieldOrVar...
```
**`segment`**
```js filename="type"
 caption | first | middle
```
**`source`**
```js filename="type"
 field | var
```
**`text`**
```js filename="type"
 @null | null | field1, field2 | VariableName or value
```
**`color`**
```js filename="type"
 field | variable | variable value
```
**`font`**
```js filename="type"
 field | variable | variable value
```
**`showLabels`**
```js filename="type"
 @null | null | true | false | yes | no
```
