## ensure

This command accepts the following arguments:

**`default`**
```js filename="type"
 invisible | read | write | writeOnce
```
**`role`**
```js filename="type"
 @null | null | Role | $AllAssistants,$AllManagers,$Assistants,$GrandManager,$Manager,$Public,$Self...
```
**`permission`**
```js filename="type"
 invisible | read | write | writeOnce
```
**`commentRoles`**
```js filename="type"
 @null | null | Role1, Role2, $AllAssistants,$AllManagers,$Assistants,$GrandManager,$Manager,$Public,$Self...
```
