## update

This command accepts the following arguments:

**`disabled`**
```js filename="type"
 @null | null | true | false | yes | no
```
**`name`**
```js filename="type"
 Symbol
```
**`label`**
```js filename="type"
 @null | @empty | label
```
**`description`**
```js filename="type"
 @null | @empty | paragraph
```
**`filterTags`**
```js filename="type"
 @null | null | text1, text2,...
```
**`indexFieldName`**
```js filename="type"
 @null | @empty | @@text | text
```
**`minRows`**
```js filename="type"
 @null | @empty | @@number | number
```
**`maxRows`**
```js filename="type"
 @null | @empty | @@number | number
```
**`asideDefaultLayout`**
```js filename="type"
 @null | null | Symbol
```
