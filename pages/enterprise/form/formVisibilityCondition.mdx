## update

This command accepts the following arguments:

**`name`**
```js filename="type"
 @null | null | Symbol
```
**`lhs`**
```js filename="type"
 @null | null | sectionField | grid/field
```
**`rhs`**
```js filename="type"
 @null | @empty | ArgBinderValue
```
**`visibilityOperator`**
```js filename="type"
 equalTo | hasNoValue | hasValue | notEqualTo
```
