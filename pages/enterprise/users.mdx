## print

This command accepts the following arguments:

**`name`**
```js filename="type"
 @null | @empty | name
```
**`handle`**
```js filename="type"
 @null | @empty | email/mobile
```
**`roles`**
```js filename="type"
 @null | null | Role1, Role2,...
```
**`manager`**
```js filename="type"
 @null | @empty | managerName
```
