## update

This command accepts the following arguments:

**`name`**
```js filename="type"
 Symbol
```
**`label`**
```js filename="type"
 @null | @empty | label
```
**`description`**
```js filename="type"
 @null | @empty | paragraph
```
**`filterTags`**
```js filename="type"
 @null | null | text1, text2,...
```
**`allowPromptAssistant`**
```js filename="type"
 @null | null | true | false | yes | no
```
**`pinnedActions`**
```js filename="type"
 @null | null | PinnedActions1, PinnedActions2,...
```
**`pinnedActionIdsForMobile`**
```js filename="type"
 @null | null | PinnedActionsMobile1, PinnedActionsMobile2,...
```
