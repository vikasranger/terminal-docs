## update

This command accepts the following arguments:

**`driveType`**
```js filename="type"
 master | partitioned
```
**`roles`**
```js filename="type"
 @null | null | Role1, Role2,...
```
**`mediaRoles`**
```js filename="type"
 @null | null | MediaRole1, MediaRole2,...
```
**`dataPartitionPeriod`**
```js filename="type"
 @null | @empty | @@dataPartitionPeriod | dataPartitionPeriod
```
**`dataRetentionDuration`**
```js filename="type"
 @null | @empty | @@duration | duration
```
**`dataUpdateDuration`**
```js filename="type"
 @null | @empty | @@duration | duration
```
**`dataPriority`**
```js filename="type"
 @null | @empty | @@dataPriority | dataPriority
```
