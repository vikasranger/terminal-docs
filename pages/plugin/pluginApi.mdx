## ensure

This command accepts the following arguments:

**`type`**
```js filename="type"
 delete | get | patch | post | put | webhook
```

## update

This command accepts the following arguments:

**`name`**
```js filename="type"
 Symbol
```
**`description`**
```js filename="type"
 @null | @empty | description
```
**`inputForm`**
```js filename="type"
 @null | null | Symbol
```
**`outputForm`**
```js filename="type"
 @null | null | Form | $FormMapOfOptions,$FormPickTree,$FormPluginConfig,$FormSetOfUser...
```
**`guaranteedInvocation`**
```js filename="type"
 @null | null | true | false | yes | no
```
**`filterTags`**
```js filename="type"
 @null | null | text1, text2,...
```

This command doesn't require any additional arguments.
